# Coming Soon Landing Page

A responsive and elegant “Coming Soon” landing page built for a premium instant chai tea brand inspired by royal Indian households.

##  Features
- Royal themed UI with warm tones
- Fully responsive design (mobile & desktop)
- Countdown timer for product launch
- Email waitlist interaction
- Smooth scroll reveal animations
- Clean and readable code structure

##  Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

##  Design Choices
The design uses warm cream, gold, and deep brown tones to reflect the heritage and elegance of traditional Indian chai culture. Minimal layout and subtle animations were chosen to maintain a premium feel while keeping the user focused on the launch message.
